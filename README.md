# Taste the Atmosphere

A Kinfolk-style café journal for slow afternoons in Bangkok & Chiang Mai.

> *Made slowly, for those who linger.*

## About

A quiet, editorial-style guide to 18 hand-picked cafés across two cities — designed in the spirit of magazine layouts with serif typography, generous white space, and atmospheric photography.

## Features

- **Hero slideshow** — five large atmospheric images, cross-fading every 6 seconds
- **Location-aware search** — pick your area (or use GPS) to find cafés within ~30 minutes
- **City grouping** — Bangkok and Chiang Mai, with neighborhood-level filtering
- **Mood filters** — work, slow, aesthetic, sweets, pets, 24-hours
- **Recommended menu** — three signature items per café, with descriptions and prices
- **Surprise me** — random pick when you can't decide

## Cafés

**Bangkok** — Roots Coffee, Factory Coffee, Hands and Heart, The Commons, Gallery Drip, Brave Roasters, Mother Roaster, Mountain Café, After You, Too Fast To Sleep, Sarnies Bangkok, Pacamara.

**Chiang Mai** — Ristr8to Coffee, Graph Cafe, Akha Ama Coffee, No.39 Café, Khagee Cafe, The Baristro at Ping River.

## Run it

It's a single self-contained HTML file — no build step, no dependencies.

```bash
open index.html
```

Or serve locally:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Stack

Vanilla HTML, CSS, JavaScript. Fonts via Google Fonts (Cormorant Garamond, Inter, IBM Plex Sans Thai, Noto Serif Thai). Imagery via Unsplash.
